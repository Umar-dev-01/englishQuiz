Discrete Mathematics - Important Questions

1. Given sets A = {2, 4, 6, 8}, B = {4, 8, 12}, find:
a. A ∪ B
b. A ∩ B
2. Write the truth table for the logical expression (p ∧ q) ∨ ¬p.
3. Determine if the relation R = {(1, 1), (2, 2), (3, 3), (1, 2)} on set A = {1, 2, 3} is:
a. Reflexive
b. Symmetric
4. How many ways can you select 2 items from a set of 5 items?
5. For a graph with vertices A, B, C, D:
a. Identify the degree of each vertex.
b. Check if the graph is a complete graph.
6. Find the greatest common divisor (GCD) of 72 and 120.
7. Find the 12th term of the arithmetic sequence where a_1 = 5 and the common difference is 3.
8. A die is rolled once. Find the probability of:
a. Rolling a number greater than 3.
b. Rolling an even number.
